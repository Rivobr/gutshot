# GUTSHOT Poker Club
## Admin Panel Architecture

Версия MVP 1.0

---

# Общая информация

Admin Panel представляет собой внутреннюю CRM-систему для управления покерным клубом.

Доступ к панели имеют только сотрудники клуба.

Панель работает через REST API Backend.

Вся бизнес-логика находится на Backend.

---

# Авторизация

Вход осуществляется по Email и Паролю.

После успешного входа Backend выдает JWT.

JWT хранится безопасно и автоматически обновляется при необходимости.

# Роли

## OWNER

Полный доступ.

Может

- управлять администраторами;
- создавать турниры;
- удалять турниры;
- изменять результаты;
- просматривать статистику;
- блокировать игроков.

---

## ADMIN

Может

- создавать турниры;
- редактировать турниры;
- открывать регистрацию;
- закрывать регистрацию;
- начинать турниры;
- завершать турниры;
- выполнять Check-In;
- просматривать игроков.

---

## MANAGER

Может

- выполнять Check-In;
- просматривать регистрации;
- просматривать Dashboard.

Изменять турниры не может.

---

# Архитектурные принципы

- Feature-Sliced Design.
- Один Widget — одна ответственность.
- Один Feature — одна задача.
- Вся бизнес-логика находится на Backend.
- Все запросы проходят через API Layer.
- Интерфейс полностью на русском языке.
- Все действия должны быть максимально быстрыми и понятными.

---

# Цель

Создать современную CRM-систему, которая позволит администраторам клуба полностью управлять турнирами, игроками и регистрациями через удобный, быстрый и безопасный интерфейс.
